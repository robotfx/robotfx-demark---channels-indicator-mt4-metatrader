<ul>
<li><strong>showBars</strong> – if = 0, the demarcation points are displayed for the entire chart</li>
<li><strong>LevDP</strong> – level of demarcation points; 2 = the central bar will be above (below) 2 bars on the left and 2 bars on the right</li>
<li><strong>qSteps</strong> – number of displayed steps, no more than 3</li>
<li><strong>BackStep</strong> – number of steps back</li>
<li><strong>startBar</strong> – if 0, recommendations for the current bar / if 1, then for the expected next bar</li>
<li><strong>TrendLine</strong> – false = there will be no trend lines</li>
<li><strong>HorizontLine</strong> – true = breakout levels will be drawn</li>
<li><strong>ChannelLine</strong> – true = build channels parallel to trend lines</li>
<li><strong>TakeLines</strong> – true = draw take levels</li>
<li><strong>Trend</strong> – 1 = UpTrendLines only, -1 = DownTrendLines only, 0 = all TrendLines</li>
</ul>
